# AI vs Human: Tic-Tac-Toe Research
This project focuses on comparing various game-playing techniques, specifically between AI algorithms and human decision-making, using the Tic-Tac-Toe game as a testing platform.

In the initial phase, a basic comparison is made between AI and human performance, where AI is shown to be faster and more accurate, while human performance is more variable. Following this, a detailed table compares the speed, intelligence, and adaptability of different methods used in the game.

The project includes a graphical representation (bar chart) which visually shows the differences, clearly indicating that:

Minimax and Alpha-Beta Pruning provide perfect decisions but lack adaptability.
Q-Learning is a learning-based approach that improves over time.
Human performance is inconsistent but shows moderate adaptability.

Additionally, the AI achievements timeline highlights the evolution of machine learning, from Alan Turing’s early ideas to modern AI systems like Deep Blue and AlphaGo.

# Overall Purpose:
The main goal of this research is to demonstrate how AI algorithms outperform humans in strategic games, not only in speed but also in accuracy and learning capabilities.
